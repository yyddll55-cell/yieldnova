## 시스템 개요

### YieldNova DeFi란?
YieldNova는 BSC(Binance Smart Chain) 기반의 분산형 포인트 플랫폼입니다. USDT를 입금하여 YieldNova 포인트로 환전하고, 바이너리 조직 구조를 통해 보너스를 획득할 수 있습니다.

### 핵심 특징
- **Web3 지갑 연동**: MetaMask, TokenPocket 등 지원
- **USDT 환전**: 실시간 환율로 포인트 변환
- **바이너리 조직도**: 상하 조직원으로부터 보너스 획득
- **레벨 시스템**: 4단계 레벨로 보너스 비율 상향
- **P2P 전송**: 회원 간 포인트 직접 전송
- **포인트 시세**: 실시간 차트로 포인트 가격 추적

## 주요 기능

### 📊 대시보드
**위치**: 메인 페이지 (/)

**표시 정보**:
- 포인트 잔액
- 현재 레벨
- 누적 보너스
- 포인트 시세
- 조직도 요약

**기능**:
- 실시간 포인트 잔액 조회
- 레벨별 보너스 정보 확인
- 빠른 메뉴 접근

### 💳 입금 페이지
**위치**: /deposit

**기능**:
1. USDT 금액 입력
2. 실시간 환율 적용
3. 보너스 분배 미리보기
4. 입금 승인 및 실행

**보너스 분배**:
- 상위 10대에게 각 9% 자동 분배
- 예: 1,000 USDT 입금 → 90 포인트가 상위 10대에게 분배

### 💸 출금 페이지
**위치**: /withdrawal

**기능**:
1. 출금할 포인트 금액 입력
2. 수수료 자동 계산 (기본 2%)
3. 수령할 USDT 금액 확인
4. 출금 요청 제출
5. 관리자 승인 대기

**출금 절차**:
- 요청 → 대기 → 승인 → 송금

### 🏢 조직도 페이지
**위치**: /organization

**기능**:
1. 바이너리 트리 시각화
2. 나를 중심으로 한 2줄 구조
3. 각 노드의 레벨 및 포인트 표시
4. 노드 확대/축소

**레벨 색상**:
- Level 1: 파란색
- Level 2: 초록색
- Level 3: 주황색
- Level 4: 빨간색

### 📈 포인트 시세 페이지
**위치**: /price-chart

**기능**:
1. 실시간 포인트 가격 차트
2. 과거 가격 추이 조회
3. 필터 (일, 주, 월)
4. 포인트 가격 변화 추적

**차트 정보**:
- X축: 날짜
- Y축: USD 가격
- 마우스 호버로 상세 정보 확인

### 🔄 P2P 전송 페이지
**위치**: /p2p-transfer

**기능**:
1. 수신자 회원 ID 입력
2. 전송 금액 입력
3. 거래 확인
4. 전송 실행

**제한사항**:
- 본인의 포인트 잔액 이상 전송 불가
- 최소 전송액: 1 포인트

## 입금 및 출금

### 입금 상세 절차

#### 사전 준비
- TokenPocket 또는 MetaMask 설치
- BSC 네트워크에 USDT 보유
- 가스비용 BNB 보유 (최소 0.01 BNB)

#### 입금 단계
```
1. 웹사이트 접속 및 지갑 연결
2. "입금" 페이지 이동
3. 입금할 USDT 금액 입력 (예: 1,000 USDT)
4. 환전될 포인트 확인 (예: 100,000 포인트)
5. 보너스 분배 확인 (상위 10대에게 9% 자동 분배)
6. "입금 승인" 버튼 클릭
7. 지갑 팝업에서 거래 승인
8. 가스비 확인 및 "승인" 클릭
9. 거래 완료 대기 (1-3분)
10. 포인트 자동 입금 확인
```

#### 입금 후 확인
- 대시보드에서 포인트 잔액 증가 확인
- 거래 내역에서 입금 기록 확인
- 보너스 수령 확인 (상위 조직원)

### 출금 상세 절차

#### 출금 단계
```
1. "출금" 페이지 이동
2. 출금할 포인트 금액 입력 (예: 10,000 포인트)
3. 수수료 확인 (기본 2% = 200 포인트)
4. 수령할 USDT 확인 (예: 98 USDT)
5. 수령 지갑 주소 확인
6. "출금 요청" 버튼 클릭
7. 거래 확인 및 제출
8. 관리자 승인 대기 (보통 24시간 이내)
9. 승인 후 USDT 수령
```

#### 출금 상태 확인
- "출금" 페이지에서 요청 상태 조회
- 상태: 대기중 → 승인 → 송금완료
- 거래 해시로 BSCScan에서 확인 가능

## 조직도 및 보너스

### 레벨 시스템

#### 레벨별 요구사항
| 레벨 | 필요 포인트 | 상위 보너스 | 하위 보너스 | 총 보너스 |
|------|-----------|----------|----------|---------|
| 1 | 30,000 | 5명 × 9% | 5명 × 9% | 최대 90% |
| 2 | 50,000 | 5명 × 9% | 5명 × 9% | 최대 90% |
| 3 | 90,000 | 5명 × 9% | 5명 × 9% | 최대 90% |
| 4 | 140,000 | 5명 × 9% | 5명 × 9% | 최대 90% |

#### 레벨 상향
- 필요 포인트 달성 시 자동 상향
- 레벨 상향 시 보너스 비율 증가
- 상위/하위 조직원의 보너스 수령 증가

### 보너스 획득

#### 보너스 종류
1. **상위 보너스**: 상위 조직원의 입금 시 9% 자동 지급
2. **하위 보너스**: 하위 조직원의 입금 시 9% 자동 지급
3. **입금 보너스**: 자신의 입금 시 상위 10대에게 9% 분배

#### 보너스 계산 예시
```
시나리오: 당신이 Level 1, 상위 5명 있음

상위 조직원 A가 1,000 USDT 입금
→ 당신이 받는 보너스: 1,000 × 9% = 90 포인트

당신이 1,000 USDT 입금
→ 상위 10대에게 각각: 1,000 × 9% = 90 포인트 분배
```

### 조직도 구조

#### 바이너리 구조
```
        [나]
       /    \
    [좌]    [우]
    /  \    /  \
  [L1] [L2] [R1] [R2]
```

- 좌측: Left 계열 조직원
- 우측: Right 계열 조직원
- 각 노드는 최대 2명의 하위 조직원 보유 가능

#### 조직도 확인
1. "조직도" 페이지 이동
2. 나를 중심으로 한 트리 확인
3. 각 노드의 레벨 및 포인트 확인
4. 노드 클릭으로 상세 정보 조회

## 포인트 거래

### P2P 전송

#### 전송 절차
```
1. "P2P 전송" 페이지 이동
2. 수신자 회원 ID 입력
3. 전송할 포인트 금액 입력
4. "전송" 버튼 클릭
5. 거래 확인
6. 전송 완료
```

#### 제한사항
- 본인 포인트 잔액 이상 전송 불가
- 최소 전송액: 1 포인트
- 최대 전송액: 본인 잔액

#### 거래 기록
- "P2P 전송" 페이지에서 거래 내역 조회
- 송금/수금 내역 확인
- 거래 날짜 및 금액 확인

### 거래 내역 조회

#### 조회 방법
1. 대시보드 또는 각 페이지에서 거래 내역 확인
2. 날짜 필터로 특정 기간 조회
3. 거래 유형별 필터 (입금, 출금, P2P, 보너스)

#### 거래 정보
- 거래 날짜 및 시간
- 거래 유형
- 금액
- 거래 상태
- 거래 해시 (BSCScan 확인 가능)

## 관리자 기능

### 관리자 대시보드
**위치**: /admin (관리자 권한 필요)

### 기능

#### 1. 포인트 가격 설정
```
1. 관리자 대시보드 접속
2. "포인트 가격 설정" 섹션 이동
3. 새로운 포인트 가격 입력 (USD)
4. "업데이트" 버튼 클릭
5. 가격 변경 완료
```

**영향**:
- USDT → 포인트 환전율 변경
- 포인트 → USDT 환전율 변경
- 모든 사용자에게 즉시 적용

#### 2. 출금 요청 관리
```
1. 관리자 대시보드의 "출금 요청" 섹션 확인
2. 대기 중인 출금 요청 목록 조회
3. 각 요청의 상세 정보 확인
4. 승인 또는 거절 선택
5. 사유 입력 (필요시)
6. 처리 완료
```

**출금 상태**:
- 대기중: 관리자 검토 대기
- 승인: 거래 진행 중
- 거절: 사유 안내
- 완료: USDT 송금 완료

#### 3. 수수료 설정
```
1. 관리자 대시보드 접속
2. "수수료 설정" 섹션 이동
3. 새로운 수수료율 입력 (%)
4. "업데이트" 버튼 클릭
5. 수수료 변경 완료
```

**적용 범위**:
- 모든 출금 거래에 적용
- 기본값: 2%
- 변경 시 신규 출금부터 적용

#### 4. 회원 관리
```
1. 관리자 대시보드의 "회원 관리" 섹션 이동
2. 회원 목록 조회
3. 검색 및 필터링
4. 회원 상세 정보 확인
5. 필요시 조치 (정지, 활성화 등)
```

**조회 정보**:
- 회원 ID
- 지갑 주소
- 포인트 잔액
- 현재 레벨
- 가입 날짜
- 거래 내역
