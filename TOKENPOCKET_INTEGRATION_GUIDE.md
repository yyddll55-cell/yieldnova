# YieldNova DeFi 시스템 - TokenPocket 연동 가이드

## 📱 TokenPocket 설치 및 설정

### 1단계: TokenPocket 앱 설치
- **iOS**: App Store에서 "TokenPocket" 검색 후 설치
- **Android**: Google Play Store에서 "TokenPocket" 검색 후 설치
- **Web**: https://www.tokenpocket.pro/ 에서 웹 버전 접속 가능

### 2단계: 지갑 생성 또는 가져오기

#### 새 지갑 생성 (처음 사용자)
1. TokenPocket 앱 실행
2. "지갑 생성" 또는 "Create Wallet" 선택
3. 지갑 이름 입력 (예: "YieldNova Wallet")
4. 비밀번호 설정 (6자리 이상, 안전한 비밀번호 권장)
5. 니모닉 백업 (12개 단어) - **매우 중요**: 안전한 곳에 기록
6. 니모닉 확인 (순서대로 입력하여 검증)

#### 기존 지갑 가져오기
1. TokenPocket 앱 실행
2. "지갑 가져오기" 또는 "Import Wallet" 선택
3. 니모닉 또는 개인키 입력
4. 네트워크 선택: **BNB Chain (BSC)**
5. 지갑 이름 설정
6. 비밀번호 설정

### 3단계: BSC 네트워크 설정

1. TokenPocket 앱에서 좌상단 네트워크 선택 버튼 클릭
2. "BNB Chain" 또는 "BSC" 선택
   - **네트워크명**: BNB Chain (BSC)
   - **Chain ID**: 56
   - **RPC URL**: https://bsc-dataseed1.binance.org:8545
3. 네트워크 전환 완료

1. YieldNova DeFi 시스템 웹사이트 방문

#### 3. 연결 승인

3. 지갑 주소 확인 및 연결 완료

#### 4. 네트워크 확인
- 자동으로 BSC 네트워크로 전환됨
- 만약 다른 네트워크로 설정되어 있다면, 수동으로 BSC(Chain ID: 56)로 전환 필요

## 💰 USDT 입금 방법

### 사전 준비
1. **TokenPocket 지갑에 BNB 보유** (가스비용)
   - 최소 0.01 BNB 권장
   - Binance 또는 다른 거래소에서 구매 가능

2. **USDT 보유**
   - BSC 네트워크의 USDT (Tether)
   - 컨트랙트 주소: `0x55d398326f99059fF775485246999027B3197955`

### 입금 절차
1. YieldNova 웹사이트 → "입금" 페이지 이동
2. "USDT 입금" 버튼 클릭
3. 입금할 USDT 금액 입력
4. 현재 환율 확인 (포인트로 변환되는 금액)
5. "입금 승인" 버튼 클릭
6. TokenPocket에서 거래 승인
   - 가스비 확인
   - "승인" 또는 "Confirm" 클릭
7. 거래 완료 대기 (보통 1-3분)
8. 포인트 자동 입금 확인

### 입금 시 보너스
- 입금 시 상위 10대에게 각각 9%씩 자동 분배
- 예: 10,000 USDT 입금 시 900 포인트가 상위 10대에게 분배

## 📊 포인트 관리

### 포인트 확인
1. 대시보드에서 "포인트 잔액" 확인
2. 현재 레벨 표시
3. 누적 보너스 현황 조회

### 포인트 출금 (USDT로 환전)
1. "출금" 페이지 이동
2. 출금할 포인트 금액 입력
3. 수수료 확인 (기본 2%)
4. 수령할 USDT 금액 확인
5. "출금 요청" 버튼 클릭
6. 관리자 승인 대기 (보통 24시간 이내)
7. 승인 후 USDT가 지갑으로 송금

### P2P 포인트 전송
1. "P2P 전송" 페이지 이동
2. 수신자 회원 ID 입력
3. 전송할 포인트 금액 입력
4. "전송" 버튼 클릭
5. 거래 완료

## 🆘 문제 해결

### 지갑이 연결되지 않음
1. TokenPocket 앱 재시작
2. 브라우저 캐시 삭제
3. 네트워크 연결 확인
4. 다른 브라우저 시도

### 거래가 실패함
1. BNB 잔액 확인 (가스비 필요)
2. 네트워크 상태 확인
3. 거래 재시도
4. 관리자에게 문의

### 포인트가 입금되지 않음
1. 거래 해시(TX Hash) 확인
2. BSCScan에서 거래 상태 확인
3. 관리자에게 거래 해시와 함께 문의

### 출금 요청이 승인되지 않음
1. 출금 요청 상태 확인
2. 필요한 서류 또는 정보 제출
3. 관리자에게 문의

## 📝 자주 묻는 질문 (FAQ)

### Q: TokenPocket 외에 다른 지갑을 사용할 수 있나요?
**A**: 네, MetaMask 등 window.ethereum을 지원하는 모든 Web3 지갑을 사용할 수 있습니다.

### Q: 포인트는 실제 화폐인가요?
**A**: YieldNova 포인트는 시스템 내부 포인트이며, USDT로 환전 가능합니다.

### Q: 레벨을 올리려면 어떻게 해야 하나요?
**A**: 필요한 포인트를 보유하면 자동으로 레벨이 상향됩니다.

### Q: 보너스는 어떻게 받나요?
**A**: 상위/하위 조직원의 입금 시 자동으로 9% 보너스가 지급됩니다.

### Q: 출금 수수료는 얼마인가요?
**A**: 기본 2%이며, 관리자가 조정할 수 있습니다.
