## 발견된 이슈 및 권고사항

### 이슈 1 (낮음): 유령 코드 Leg View 필터링 미구현
- **현상:** `isGhost` 플래그 없어 관리자 가상 노드가 일반 유저 Leg View에 표시됨
- **권고:** `users` 테이블에 `isGhost BOOLEAN DEFAULT FALSE` 컬럼 추가 후 `getOneLineLegMembers` 쿼리에 `WHERE isGhost = FALSE` 조건 추가

### 이슈 2 (정보): 10대 기부 보너스 gen10 미수령
- **현상:** gen10이 직추천인이므로 기부 체인은 gen9부터 시작, gen10은 기부 보너스 미수령
- **원인:** 정상 동작 — 기부 보너스는 직추천인의 상위(gen9~gen1)에게 분배됨

### 이슈 3 (정보): 레벨 미보유 직추천인 컷오프
- **현상:** gen10(레벨0)이 레벨2 구매자(A)의 직추천인일 때 보너스 admin 귀속
- **원인:** 정상 동작 — 설계된 컷오프 로직
