## 주요 관리 기능

### 1. 포인트 가격 설정

#### 목적
- USDT ↔ 포인트 환전율 관리
- 시장 상황에 따른 가격 조정
- 일별 포인트 가격 기록

#### 설정 방법
```
1. 관리자 대시보드 접속
2. "포인트 가격 설정" 섹션 이동
3. 현재 포인트 가격 확인
4. 새로운 가격 입력 (USD)
   예: 0.01 USD/포인트
5. "업데이트" 버튼 클릭
6. 변경 사항 저장
```

#### 가격 설정 예시
| 날짜 | 포인트 가격 | 환전율 | 비고 |
|------|-----------|--------|------|
| 2026-05-12 | $0.01 | 1 USDT = 100 포인트 | 초기 가격 |
| 2026-05-13 | $0.012 | 1 USDT = 83.33 포인트 | 상향 조정 |
| 2026-05-14 | $0.011 | 1 USDT = 90.91 포인트 | 조정 |

#### 가격 설정 시 고려사항
- 시장 수요 및 공급
- 경쟁사 가격 비교
- 사용자 만족도
- 플랫폼 수익성

#### 영향 범위
- 신규 입금 시 적용
- 신규 출금 요청 시 적용
- 기존 거래에는 영향 없음

### 2. 출금 수수료 설정

#### 목적
- 출금 거래 수수료 관리
- 플랫폼 운영비 확보
- 사용자 비용 투명성

#### 설정 방법
```
1. 관리자 대시보드 접속
2. "출금 수수료 설정" 섹션 이동
3. 현재 수수료율 확인
4. 새로운 수수료율 입력 (%)
   예: 2% (기본값)
5. "업데이트" 버튼 클릭
6. 변경 사항 저장
```

#### 수수료 계산 예시
```
출금 요청: 10,000 포인트
수수료율: 2%
수수료: 10,000 × 2% = 200 포인트
순 출금액: 10,000 - 200 = 9,800 포인트
```

#### 권장 수수료율
- 최소: 1% (낮은 수수료)
- 표준: 2% (권장)
- 최대: 5% (높은 수수료)

### 3. 출금 요청 관리

#### 출금 요청 확인
```
1. 관리자 대시보드 접속
2. "출금 요청" 섹션 이동
3. 대기 중인 출금 요청 목록 조회
4. 각 요청의 상세 정보 확인
```

#### 출금 요청 정보
| 항목 | 설명 |
|------|------|
| 회원 ID | 출금 신청자 ID |
| 포인트 금액 | 출금 요청 포인트 |
| USDT 금액 | 환전될 USDT 금액 |
| 수수료 | 차감될 수수료 |
| 신청일 | 출금 요청 날짜 |
| 상태 | 대기중/승인/거절/완료 |

#### 출금 승인 절차
```
1. 출금 요청 목록에서 요청 선택
2. 상세 정보 확인
   - 회원 정보 검증
   - 포인트 잔액 확인
   - 지갑 주소 확인
3. 승인 조건 확인
   - 포인트 충분 여부
   - 사용자 신원 확인
   - 거래 이상 여부
4. "승인" 버튼 클릭
5. 거래 실행
   - USDT 송금
   - 포인트 차감
   - 거래 기록 저장
6. 사용자에게 알림 발송
```

#### 출금 거절 절차
```
1. 출금 요청 목록에서 요청 선택
2. 거절 사유 확인
   - 포인트 부족
   - 신원 미확인
   - 의심 거래
   - 기타 사유
3. "거절" 버튼 클릭
4. 거절 사유 입력
5. 사용자에게 알림 발송
6. 포인트 반환 (자동)
```

#### 출금 상태 추적
| 상태 | 설명 | 다음 단계 |
|------|------|---------|
| 대기중 | 관리자 검토 대기 | 승인/거절 |
| 승인 | 거래 진행 중 | 송금 완료 |
| 거절 | 출금 거절됨 | 재신청 가능 |
| 완료 | USDT 송금 완료 | 종료 |

### 4. 회원 관리

#### 회원 정보 조회
```
1. 관리자 대시보드 접속
2. "회원 관리" 섹션 이동
3. 회원 목록 조회
4. 검색 또는 필터링
```

#### 회원 정보 항목
| 항목 | 설명 |
|------|------|
| 회원 ID | 고유 식별자 |
| 지갑 주소 | Ethereum 지갑 주소 |
| 포인트 잔액 | 현재 보유 포인트 |
| 레벨 | 현재 레벨 (1-4) |
| 가입일 | 회원가입 날짜 |
| 마지막 거래 | 최근 거래 날짜 |
| 상태 | 활성/정지 |

#### 회원 상세 정보 확인
```
1. 회원 목록에서 회원 선택
2. 상세 정보 페이지 이동
3. 다음 정보 확인:
   - 기본 정보 (ID, 지갑, 이름)
   - 포인트 정보 (잔액, 누적 입금, 누적 출금)
   - 레벨 정보 (현재 레벨, 진행도)
   - 조직도 정보 (상위, 하위 조직원)
   - 거래 내역 (입금, 출금, P2P, 보너스)
```

#### 회원 상태 관리
```
활성 회원 → 정지 (필요시)
1. 회원 상세 정보 페이지 이동
2. "상태 변경" 버튼 클릭
3. 정지 사유 입력
4. "정지" 버튼 클릭
5. 회원 정지 처리

정지 회원 → 활성화
1. 회원 상세 정보 페이지 이동
2. "상태 변경" 버튼 클릭
3. "활성화" 버튼 클릭
4. 회원 활성화 처리
```
